## Задание 17328

```
import itertools
alphabet = "ГЕРАСИМ"
con = 'ГРСМ'
vol = 'ЕАИ'
ar = itertools.permutations(alphabet)
arl = []
for i in ar:
    arl.append(list(i))
count = 0
for e in arl:
    flag = True
    for i in range(len(e) - 1):
        if (e[i] in con and e[i + 1] in con) or (e[i] in vol and e[i + 1] in vol):
            flag = False
    if flag == True: count += 1
print(count)
```
* `Ответ: 144`
